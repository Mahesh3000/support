# mahesh_support (Frontend)

A small frontend project containing essential utilities and components for support workflows.

## Prerequisites

- Node.js (>= 14)
- npm or yarn

## Setup

1. Clone the repo:
   git clone <repo-url>
2. Install dependencies:
   npm install
   // or
   yarn install

## Run (development)

npm start
// or
yarn start

Open http://localhost:3000

## Build

npm run build
// or
yarn build

## Project structure

- src/ — source code
  - components/ — reusable components
  - pages/ — application pages
  - styles/ — styles and assets
- public/ — static files
